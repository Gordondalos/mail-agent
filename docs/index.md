Навигация по документации

- `docs/` — документация (этот файл и дополнительные страницы)
- `release/` — собранные артефакты
- `scripts/` — утилиты для разработки и сборки (например, `setup.ps1`, `build.js`)
- `frontend/` и `frontend_legacy_dom/` — фронтенд-ресурсы и старые версии
- `src-tauri/` — Rust код Tauri (интеграция с ОС, трей, нативные операции)
- `src/` — исходники фронтенда (Angular приложение)
Структура проекта (основные директории):

- Цель: информировать пользователя о новых письмах и дать быстрый доступ к письмам из системного трея.
- Технологии: Angular (frontend), TypeScript, Tauri + Rust (native), Node.js и npm (скрипты и сборка), Cargo (Rust).
Кратко:

Gmail Tray Notifier — кроссплатформенное Tauri-приложение, которое живёт в системном трее, проверяет непрочитанные письма Gmail и показывает кастомное полупрозрачное всплывающее окно (tray popup) с быстрыми действиями.
