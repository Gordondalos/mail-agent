# Разработка и архитектура

Краткое руководство для разработчиков: как запустить, отладить и собрать проект.

Структура кода
- `src/` — Angular приложение (компоненты, сервисы, стили)
- `src-tauri/` — Rust-часть: `main.rs`, `oauth.rs`, `notifier.rs`, `gmail.rs`, `config.rs`
- `scripts/` — утилиты для облегчения сборки и упаковки

Запуск в режиме разработки
1) Установите зависимости: `npm install` и Rust toolchain
2) Запустите фронтенд в режиме наблюдения: `npm run start` (или `npm run dev` в зависимости от package.json)
3) Запустите Tauri в режиме разработки или нативной отладки (см. `scripts/dev.js`)

Сборка и тестирование
- Сборка фронтенда: `npm run build`
- Сборка Tauri: `npm run build:windows` / `npm run build:linux`

Отладка
- Логи Tauri можно смотреть в выводе консоли при запуске `cargo run` или через инструменты разработчика в веб-просмотре.
- Для проблем в TypeScript — используйте `ng serve` или `npm run start` для быстрого перезапуска.

Контрибьюция
- Ведите изменения в отдельных ветках и открывайте pull request с описанием изменений.
- Добавляйте тесты и документацию для значительных изменений.
