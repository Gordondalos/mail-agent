# Быстрая инструкция: Развёрнутый просмотр

## 🖱️ Как использовать

### Просмотр письма в 3 клика:

```
1. Пришло письмо
   ┌─────────────────────────────────┐
   │ ✉️  Новое письмо от Gmail       │
   │                                  │
   │ От: john@example.com             │
   │ Тема: Важная информация          │
   │ Snippet: Привет! У меня есть...  │
   │                                  │
   │  [Перейти] [Прочитано] [×]      │
   └─────────────────────────────────┘
         ↓
         
2. 🖱️ Двойной клик по окну
   ┌────────────────────────────────────────────┐
   │ ✉️  Новое письмо от Gmail                  │
   │                                             │
   │ От: john@example.com                        │
   │ Тема: Важная информация                     │
   │ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
   │                                             │
   │ Привет!                                     │
   │                                             │
   │ У меня есть для тебя важная информация...   │
   │ [HTML содержимое письма с форматированием]  │
   │ [Изображения, ссылки, таблицы]             │
   │                                             │
   │ [Прокрутка для длинных писем] ↕️            │
   │                                             │
   │  [Перейти] [Прочитано] [×]                 │
   └────────────────────────────────────────────┘
         ↓
         
3. 🖱️ Двойной клик снова = свернуть обратно
```

## ⚙️ Настройка размеров

### В настройках приложения:

```
Размеры окна уведомлений
┌────────────────────────────────────────┐
│ Ширина окна (px):        [650      ]   │ ← Обычный размер
│ Высота окна (px):        [150      ]   │
└────────────────────────────────────────┘

Размеры развёрнутого окна (двойной клик)
┌────────────────────────────────────────┐
│ Ширина развёрнутого:     [800      ]   │ ← При двойном клике
│ Высота развёрнутого:     [600      ]   │
└────────────────────────────────────────┘

            [Сохранить настройки]
```

## ✨ Что показывается

### В свёрнутом виде:
- ✅ Тема письма
- ✅ Дата и время
- ✅ От кого (с email)
- ✅ Кому (с email)  
- ✅ Краткое содержание (snippet)

### В развёрнутом виде:
- ✅ Всё из свёрнутого (кроме snippet)
- ✅ **Полное содержимое письма**
- ✅ HTML форматирование
- ✅ Изображения
- ✅ Кликабельные ссылки
- ✅ Прокрутка для длинных писем

## 🎯 Советы

💡 **Двойной клик работает везде**, кроме кнопок  
💡 **Окно можно перетаскивать** в обоих режимах  
💡 **Настройки сохраняются** автоматически  
💡 **Размеры можно менять** от 400×300 до 1920×1080

## 🐛 Решение проблем

**Q: Окно не разворачивается?**  
A: Кликайте быстро дважды, не на кнопках

**Q: Содержимое не показывается?**  
A: Некоторые письма могут не иметь тела. Используйте "Перейти"

**Q: Email показывается с &lt; и &gt;?**  
A: Это исправлено! Должно показываться: `John <john@example.com>`

**Q: "Кому:" сдвинуто вправо?**  
A: Это исправлено! Теперь выровнено с "От:"
